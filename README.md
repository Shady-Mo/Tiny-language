# Tiny-language

Tiny-language is a C++ project aimed at showcasing or implementing features related to a custom programming language or compiler. This project provides users with a simple yet powerful way to experiment with language design, parsing, or related systems.

## Features

- **Custom Syntax**: Define and parse your own programming language syntax.
- **Extensible Design**: An architecture that supports adding new language features.
- **Efficient Parsing**: Leverages C++ for high-performance parsing and analysis.

## Getting Started

### Prerequisites

To build and run this project, ensure you have:

- A C++17 (or later) compatible compiler.
- CMake (version 3.15 or higher recommended).
- A supported operating system (Linux, macOS, or Windows).

### Building the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/Shady-Mo/Tiny-language.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Tiny-language
   ```
3. Create a build directory and generate the build system with CMake:
   ```bash
   mkdir build && cd build
   cmake ..
   ```
4. Compile the project:
   ```bash
   cmake --build .
   ```

### Running the Project

After building, you can execute the main program:
```bash
./TinyLanguageExecutable
```
